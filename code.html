                    //get the values from the UI
                    //starts or controller function
                    function getValues() {
                        //get the values from the page
                        let startValue = document.getElementById("startValue").value;
                        let endValue = document.getElementById("endValue").value;

                        //We need to validate our input
                        //Parse into Integers
                        startValue = parseInt(startValue);
                        endValue = parseInt(endValue);
                        
                        //Check if the numbers are int, if not alert them to enter integers
                        if (Number.isInteger(startValue) && Number.isInteger(endValue)){
                            //we call generateNumbers
                            let arrayNumbers = generateNumbers(startValue, endValue);
                            //we call displayNumbers
                            displayNumbersAndBoldEven(arrayNumbers);
                        } else {
                            alert("You must enter integers");
                        }

                        
                    }

                    //Generate numbers from the startValue to the endValue
                    //logic function
                    function generateNumbers(startValue, endValue) {

                        let numberValues = []
                        
                        for (let i = startValue; i <= endValue; i++) {
                            numberValues.push(i);
                        }
                        return numberValues;
                    }

                    //Display all the numbers and mark the even numbers bold
                    //display or view function
                    function displayNumbersAndBoldEven(numbers) {

                        let table = document.getElementById("results");
                        table.innerHTML = "";

                        let className = "";

                        numbers.forEach(element => {
                            if (element % 2 === 0 && element !== 0) {
                                className = "even";
                            } 
                            else {
                                className = "odd";
                            }
                            table.innerHTML += `<tr><td class="${className}">${element}</td></tr>`;
                        });
                    }

The Code is structured in three functions. The code is made in ES6.

ZHERO

ZHERO is a program which displays the numbers choosen by the user. If no number is chosen, it will display the default numbers (0-100).
The app is executed by three functions. Each function has its porpuse:

getValues()

This function controls and starts the application. First, it gets the values from the user (or the default ones) and assigns it into two variables startValue and endValue. Then, we change the values to integers calling the function parseInt() and validate if the values are numbers. To avoid any possible bug, we have put in place and if/else statement for this purpose, but the HTML element from where the values are collected also has the type number. If the validation state has passed, we call the logic function and store its data in a variable and we pass this data to the display function to show the result to the user.

generateNumbers(startValue, endValue)

This function returns an array of numbers between the startValue and the endValue. Both have to be passed to the function by putting them in its arguments. First, we create a variable and we iniciate it declaring an empty array. Then we loop between the two values and store the values passed and between them into the variable. Once it has ended the loop, it returns the array.

important!

There is there are two variables with the same namesame name as long as the scope of the old variable does not interfere with the new one one. For example: There are two variables but within the scope of two different functions.

displayNumbersAndBoldEven(numbers)

The main purpose of this function is to display in the page the numbers between the ones it has chosen the user, counting them. First, we declare a variable table and assign it to the element with the id "results". Then the inner HTML is reseted to only display the range of numbers chosen, besides we crate another variable to control if the number is even or odd. After, we loop from the array of numbers passsed to the function and we check if it is even. If it is we assign the name "even" to the last variable, if not the name "odd". Finally, we sum to the inner HTML of the table element a row and a cell of a table with the classname which it has to have with its number.
